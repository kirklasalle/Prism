## 🚀 Quick Start

### Installation

1. **Environment Setup**

   ```bash
   cd .mcp/impressioncore-ipa
   pip install -r requirements.txt
   ```

2. **MCP Configuration**
   - The Ultimate Edition is automatically configured in `mcp-settings.json`
   - Server name: `impressioncore-ipa-ultimate`
   - Entry point: `server_ultimate.py`

### Basic Usage

```python
# Ultimate research with hybrid methodology
result = await ipa_ultimate_research({
    "query": "Latest developments in AI safety research",
    "methodology": "hybrid",
    "depth_level": 4,
    "real_time": true
})

# Conversational search
response = await ipa_conversational_search({
    "query": "How does quantum computing affect encryption?",
    "context": "Previous discussion about cybersecurity"
})
```
