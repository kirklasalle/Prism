  2. Deterministic pass/fail semantics.
  3. Traceable artifacts and signatures.
  4. Policy-aware behavior validation.
  5. Public-safe demo path.

Gate architecture

  • Gate 0: doctor + integrity
  • Gate 1: build/type/contracts
  • Gate 2: core unit/integration
  • Gate 3: governance behavior
  • Gate 4: PTAC fast deterministic
  • Gate 5: demo qualification
  • Gate 6: strict release/security signatures
