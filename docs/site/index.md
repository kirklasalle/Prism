# PRISM

> **Agents-as-a-Service infrastructure** — autonomous agents as deployable, scalable, orchestratable services with strong governance primitives.

Welcome to the PRISM external documentation site.

## What is PRISM?

PRISM is an open-core platform for building and operating production-grade AI agents. It treats agents as first-class deployable services — with audit trails, approval queues, multi-tenant isolation, and policy-aware routing built into the fabric.

## Highlights

- **Spectrum Refraction (SR)** — parallel multi-hemisphere LLM generation with consensus aggregation. See [PAD_WHITEPAPER.md](../PAD_WHITEPAPER.md).
- **Character Accountability Chain (CAC)** — operator → character → session traceability with OAuth-bound identity verification.
- **Constitutional Causal Compiler (CCC)** — auto-compile policy + memory + workflow constraints into enforceable runtime plans.
- **Profile-aware governance** — Individual / Business / Enterprise modes with distinct policy posture, approval thresholds, and audit retention.
